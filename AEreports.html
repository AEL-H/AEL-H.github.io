Laboratory Reports

Laboratory Reports First-year
Geometrical optics Joint laboratory report. The focus of the experiment was to develop skills in determining the focal length of different lenses experimentally.
Thermal radiation Individual laboratory report. The aim of this experiment was to use calibration data effectively to experimentally verify radiation laws.
Laboratory Reports Second-year
Thermal diffusivity Joint laboratory report. The aim of this experiment was to obtain a value for the thermal diffusivity of an epoxy resin rod. This experiment involved varying the the surface temperature of the rod abruptly and periodically.
Cornu's method Individual laboratory report. The aim of this experiment was to experimentally obtain elastic constants of a glass bar using optical interference. We extended the laboratory script by introducing a raspberry pi to take our measurements and analysed this data using Wolfram Mathematica.
Laboratory Report Third-year
Matter/Antimatter asymmetries at the LHC Individual short-form laboratory report. The aim of this experiment was to try find evidence for CP violation using data from the 2011 run of LHCb. This was done using the software framework ROOT in C/C++.

Theory Computing Project

The Ising Spin Model
The Ising spin model Joint report. The report details our theory computing project where we simulated the Ising Spin Model(ISM) using python. In particular we were interested in the behaviour of the ISM in different geometries including fractional dimensions in addition to 1D, 2D and 3D.
Investigation into Numerical Integration
Largely based upon notes provided by Niels walet
Week 1: Oscillators An investigation into oscillator behaviour using numerical techniques, most notably the RK4 method, to solve differential equations.
Week 2: Monte Carlo An investigation into numerical integration and importance sampling. Both normalised and unnormalised distributions were investigated.
Week 3: Matrix methods An investigation into the use of matrix methods to solve a set of coupled equations.

Code produced for PHYS 20161: Introduction to Programming for Physicists (15/16):
Python Assignment Python assignment aimed at visualisation of data. The assignment was open-ended and we were tasked to visualise geographical data contained within a csv file. I used Bokeh to create an interactive html document where the cities were displayed on google maps. Additional information could be seen by hovering over cities. I also used this assignment to get to grips with GUI development.
C++ Assignment C++ assignment aimed at optimisation. The assignment was open-ended and we were tasked to read in geographical data from a csv file and determine the optimum place to create a hub connecting the locations. I looked at a few search heuristics including simulated annealing and attempted a simplified implementation of the travelling salesman problem.
C++ Assignment[optional] An optional assignment to create a C++ program that could solve for the inverse matrix using gaussian elimination. I used the opportunity to explore C++ operator overloading and to develop skills in creating structs.

Code produced for PHYS 20872: Theory Computing Project (15/16):
Ising spin model Contains a mix of python files and matlab macros. These were the main codes used to simulate the model, obtain the relevant quantities and perform the necessary data analysis.
